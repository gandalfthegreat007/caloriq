## ✨ Features

- **📝 Daily Food Log** — search 65+ Indian foods, set quantity, track instantly
- **📊 Dashboard** — day / week / month calorie & protein charts
- **⚖️ Weight Tracker** — log weight daily, see actual vs expected loss curve
- **🎯 Deficit Tracker** — know if you're actually losing 0.45kg/week or not
- **🧠 BMI + TDEE** — Mifflin-St Jeor formula, scientifically proven targets
- **🥗 Food Library** — edit existing foods or add your own
- **💾 Offline** — all data saved in browser localStorage, no account needed

## 📐 Science Behind It

| Metric | Formula |
|---|---|
| BMR | Mifflin-St Jeor |
| TDEE | BMR × 1.375 (light activity) |
| Safe deficit | −500 kcal/day |
| Expected loss | ~0.45 kg/week |
| 1 kg fat | 7,700 kcal |
